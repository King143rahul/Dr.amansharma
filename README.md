# Dr. Aman Sharma

Academic portfolio website for Dr. Aman Sharma, PhD, Assistant Professor of Chemistry in Bengaluru and founder of AMSH Endeavours.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Vercel-ready SPA routing

## Scripts

```bash
npm install
npm run dev
npm run build
npm run lint
```

## Deploy

This project is ready for Vercel.

- Framework Preset: `Vite`
- Build Command: `npm run build`
- Output Directory: `dist`
